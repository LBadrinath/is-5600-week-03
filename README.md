# Lab 3 - Node API

This project includes:

- A basic Node.js API using the built-in `http` module
- A simple Express chat API

## Run the HTTP server

```bash
npm run start:http
```

## Run the Express server

```bash
npm run start:express
```

## HTTP Server Routes

- GET /
- GET /about
- GET /users

## Express Server Routes

- GET /
- GET /messages
- GET /messages/:id
- POST /messages
- DELETE /messages/:id
